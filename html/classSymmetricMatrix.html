Detailed Description

template<typename T>
class SymmetricMatrix< T >

@SymmetricMatrix Class to hold arbitrarily big symmetric square matrices. For a matrix of size NxN, only Nx(N+1)/2 elements are stored.

bool SymmetricMatrix< T >::TestDistDisMat

Test of correctness
This is meant to test if the symmetric matrix is a distance or dissimilarity matrix. It checks that all elements in the main diagonal are 0 and all outside the main diagonal are stricty possitive

Returns
