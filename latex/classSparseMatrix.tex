\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Sparse\+Matrix$<$ T $>$}

@\+Sparse\+Matrix Class to hold arbitrarily big sparse matrices. Elements are stored with column index + value in a vector associated to each row.~\newline
 Time to set and get elements are of order O(log\+\_\+2(\+Nc)) being Nc the number of columns.~\newline
 Space is O(N$\ast$(sizeof(element)+sizeof(index)), being element the type of the matrix contents and index that of the matrix index (which is currently unsigned int).
